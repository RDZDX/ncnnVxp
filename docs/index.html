IPoverObexVxp

A client application connects via Bluetooth to a server application for the MRE platform on mobile phones, including Nokia S30+ devices. When a Bluetooth connection is established, the phone application automatically switches to chat mode. On the server, start the Bluetooth service and run: sudo ./IPoverObexPC After the connection becomes active, open an additional server console and run: nc 127.0.0.1 400 The phone application can be switched to local mode (for console commands) by pressing * + [Swch]. Before use, the phone and server must be paired and marked as trusted Bluetooth devices. For use on Nokia mobile phones, the application must be signed using the IMSI code of your SIM card. More information is available at: https://vxpatch.luxferre.top "IPoverObexTest.vxp" "IPoverObexPC". [32 bit binary for Raspberry Pi Zero W Debian Bookworm (12)].
